# Customer Churn Analysis Dashboard (Excel Project)

## Project Overview
This project analyzes customer churn data using Excel. The goal of this project is to identify how many customers left the company and calculate the churn rate. A dashboard was created using pivot tables, charts, and KPI cards.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Charts
- Data Cleaning

## Key Metrics
- Total Customers: 7043
- Churn Customers: 1869
- Churn Rate: 26.5%

## Process
1. Cleaned the raw data.
2. Created pivot tables.
3. Calculated churn rate.
4. Created dashboard using charts and KPI cards.

## Conclusion
The churn rate is 26.5%, which means many customers are leaving the company.
